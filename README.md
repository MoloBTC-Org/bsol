## Overview
BSOL v1.0 is the canonical permissive license for Bitcoin-related code, papers, analyses, curricula, designs, reference implementations, and knowledge works. It emphasizes radical freedom of use (“take it, fork it, use it freely”), explicit fork rights, sovereign protections (pseudonymity, privacy, free-speech safe harbour), and a living governance model.

The **Sovereign Bitcoin Knowledge Declaration v1.0** is the primary manifesto-style document released under BSOL. It articulates principles of non-rivalrous knowledge, the unbroken lineage from cypherpunks to Bitcoin, operational discipline for sovereign nodes, and practical resources for verification, privacy, and self-hosting.
